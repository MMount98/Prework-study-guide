CSS

  • Stands for Cascading Style Sheet
  • Code worked on in files ending in .css
  • Used to make webpage presentable
  • CSS breaks down to internal, external, and inline CSS
  • Internal CSS is used for styling three to four emlemnts at a time, it is held in the head of the HTML
  • External CSS is used to bring in external CSS files and held in the head of HTML
  • Inline CSS for single line styling within any tag
  • Declarations are the core of CSS. They apply a style to the selected element
  • Selector—The beginning of the ruleset used to target the element that will be styled
  • Declaration Block—The code in-between (and including) the curly braces ({ }) that contains the CSS declaration
  • Declaration—The group name for a property and value pair that applies a style to the selected element
  • Property—The first part of the declaration that signifies what visual characteristic of the element is to be modified
  • Although CSS is a different language than HTML, its possible to write CSS code directly within HTML code using inline styles but it is not commly used
  • All HTML elements are boxes made up of four components: a content container, padding, border, and margin
  • padding: The amount of space between the content area and the border
  • border: The thickness and style of the border surrounding the content area and padding
  • margin: The amount of space between the border and the outside edge of the element
  • CSS style files need to be connected to the index with code in the head
  • Margin indicates how much space we want around the outside of the element
  • Padding indicates how much room we want around the content inside the element
  • Most css lines need to ended with a semicolin
  • CSS is used to give orginzation and color to the website
  • CSS includes properties that change how a browser positions elements
  • We can fix an element to a specific position on the page (regardless of user scrolling) by setting its position to fixed
  • Foreground color is the color that an element appears in
  • Conversely, when a heading is styled so that its background appears yellow, the background color of the heading has been styled
  • The hexadecimal and rgb color system can represent many more colors than the small set of CSS named colors
  • There is another equally powerful system in CSS called the hue-saturation-lightness color scheme, abbreviated as HSL
  • To use opacity in the HSL color scheme, use hsla instead of hsl, and four values instead of three
  • The RGB color scheme has a similar syntax for opacity, rgba
  • When an elements position is set to absolute, as in the last exercise, the element will scroll with the rest of the document when a user scrolls
