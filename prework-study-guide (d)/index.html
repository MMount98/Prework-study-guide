HTML

  • HTML stands for Hypertext Markup Language
  • HTML cannont be used in a CSS file
  • A simplier language that isn't case-sensitive
  • Carries display information for the webpage
  • Runs from the index in the files of a repo
  • Useful for adding images and videos to a webpage
  • The head element contains informatio about the webpage
  • The body element represents the visual aspect for the user
  • Unlike in Javascript the semicolin isn't used in HTML
  • HTMl is used to create the framework of th website itself (Javascript and CSS allows elemnts to be used on the site)
  • The use of the aligator brackets are necessarcy for writing in HTML

CSS

  • Stands for Cascading Style Sheet
  • Code worked on in files ending in .css
  • Used to make webpage presentable
  • CSS breaks down to internal, external, and inline CSS
  • Internal CSS is used for styling three to four emlemnts at a time, it is held in the head of the HTML
  • External CSS is used to bring in external CSS files and held in the head of HTML
  • Inline CSS for single line styling within any tag
  • CSS style files need to be connected to the index with code in the head
  • Margin indicates how much space we want around the outside of the element
  • Padding indicates how much room we want around the content inside the element
  • Most css lines need to ended with a semicolin
  • CSS is used to give orginzation and color to the website

GitHub

  • Github is an online site used to store and hold Repos
  • Github allows for many coders to work and write on the same code at the same time
  • Pull system allows for mulitple editors to double check ones work and safelty merge code
  • Github will take the diffrent branches and merge them together well checking all the new code to make sure it all flows
  • Allows for the possiblity for world-wide collaboration with multiple developers
  • Github is great for preventing the main code from becoming broken
  • Creates one safe location for huge projects to hold all thier files
  • Github is free website that main diffrent compaines
  • Tracks file changes on a local system and makes it easier to track errors
  • Has a VCS to protect all the diffent versions of the main code incase somehting breaks

Javascript

  • Javascript can add interactivity to the website
  • Script tag is used for JS in HTML
  • An Error code in JS will stop the whole process
  • Applied to HTML
  • Saved in a .js file
  • Javascript can be used to create animations
  • Can be used server-side and user-side
  • Considered to be the language to create the first dynamic webpages
  • Functions in JS ar consider objects
  • Javascript is linked to the HTMl under footer section
  • Javascript is used to make the website fully functional
  • Javascript is kind of like the behind the scenes to a website
